#  Online Book Store

##  About Project
Online Book Store is a web-based application developed for students to browse, purchase, and download books online. It provides an easy-to-use interface and efficient database management.

##  Features
- User Registration & Login
- Book Browsing and Search
- Book Ordering System
- Purchase Bill Genrating
- Free PDF Download Option

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: JSP
- Database: Oracle SQL

##  How to Run
1. Clone the repository  
2. Import database using Oracle SQL  
3. Run project on Apache Tomcat server  
4. Open in browser: http://localhost:8080/onlinebookstore/
